# Space Webgame
A tiny space themed survival &amp; shooting game made with HTML, CSS, JS

The goal is to score the highest, by surviving the longest and destroying the asteroids with a laser beam.


##__How to play : 

* The first game will start on its own when the page loads.
* The player can pause/resume the game, or restart it from 0 with the reset button. 

- The spaceship will move from left to right by default, but the player can switch directions by pressing the d key.
- The player can destroy the asteroids with a laser beam. Pressing the space key will allow to aim, releasing it will shoot in the desired direction.
- The speed can be modified with the slider on the bottom of the page. The higher the speed, the higher the points for 'survival'.

- Crashing the spaceship into an asteroid will end the game.
